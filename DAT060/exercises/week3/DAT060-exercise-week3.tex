$S(x) : x $ is student,\\
$P(x) : x $ is a problem,\\
$R(x, y) : x $ solves $y$\\\\

$\exists x (S(x) \land( \forall y (P(y) \ra R(x,y))))$\\\\

\textbf{a) $\forall x\forall y R(x,y) \vdash \forall x R(x,x)$}\\

proof:
	\[
	\Jproof{

	}
	\]

\textbf{b) $\forall x (P(x) \ra S) \vdash (\exists x P(x)) \ra S$}\\

proof:
	\[
	\Jproof{

		}

	}
	\]
